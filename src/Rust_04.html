Estrutura de controle

As variáveis são elementos básicos na programação, definimos uma variável como espaços na memória ao qual somos capazes de armazenar e representar dados.
Os dados dentro de uma variável podem ter diferentes tipos. Como números inteiros, textos e valores como verdadeiro e falso.

variáveis podem ser classificadas como globais e locais.

Variáveis globais são aquelas declaradas fora do escopo das funções e podem ser utilizadas por todo o projeto.

Variáveis locais são aquelas declaradas no início do bloco e seus escopos são restritos aos blocos em que foram declaradas.

como usar variáveis: 

      
    fn main() { 

    let x = 45; 

    println!("O valor é {}", x);  

    }

Nota de Sintaxe:

  • let em rust indica a criação de uma variável.
  • X representa o nome da nossa variável, também chamadas de identificadores.
  • 45 é o valor atribuído para a variável x
Como resultado para esse exemplo temos o valor de X exibido no terminal: "O valor é 45"
Seguindo com os exemplos, vamos alterar o valor de X 

      
    fn main() { 

        let mut x = 45; 

        println!("O valor é {}", x); 

        x = 60; 

        println!("O valor é {}", x); 

    }

Resultado para o código:
O valor é 45
O valor é 60

Como podemos observar foi adicionado ao código a [palavra-chave] ‘mut’.

Mut – em rust, indica que a variável é flexível e que pode ter seu valor alterado.

Para a variável x com o valor 60 atribuído, vemos que não foi utilizado o ‘let’, isto ocorre porque a variável já havia sido criada, desta forma não é necessário usar a palavra-chave ‘let’ novamente, apenas o identificador da variável.

Nota: [palavra-chave] trata-se de uma palavra ou identificador que tem um significado especial para a linguagem.
