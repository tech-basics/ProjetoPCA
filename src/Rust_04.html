Estrutura de controle

Na area de computação, estruturas de controle referem-se a ordem em que instruções e funções são executadas ao verificar se alguma condição é verdadeira ou falsa.
Por meio dessa verificação de verdadeiro ou falso as estruturas de controle possibilitam ao programa tomar decisões e alterar o seu fluxo de execução.

If ✧ Else If ✧ Else

O comando [if] é comum em muitas linguagens de programação, sua função é verificar se uma condição é verdade ou falsa.
Assim como demostrado no exemplo a seguir:

      
let age = 13;

if age < 18 {
    println!("Olá, criança"); // O código mostra na tela
}

Complementar ao [if] ainda temos os operadores [else if] e [else]. Essa função possibilita adicionar uma nova condição à estrutura de controle para atender a lógica implementada. 

      
let i = 7;

if i % 2 == 0 {
    println!("par");
} else {
    println!("ímpar"); // O código mostra na tela
}

Para o [else if] utilizaremos para operações maiores, com mais caminhos para o código.
Outro exemplo:

      
let tam_equipe = 7;

if tam_equipe < 5 {
    println!("equipe pequena");
} else if tam_equipe < 10 {
    println!("equipe média"); 
} else if tam_equipe == 2 {
    println!("dupla");
} else {
    println!("equipe grande");
}

Também é possível definir uma váriavel utilizando let e dentro dela estabelecer uma função de verificação.
Como mostrado no exemplo a seguir:

      
let idade: u8 = 13;
let menor_idade = if idade < 18 { true } else { false };

While

Assim como a função anterior [while] ou "Enquanto" só será possível quando for atendido uma condição.
Para ele "enquanto" uma condição for verdadeira, o código será executado, e enquanto essa condição permanecer verdadeira, o código continua sendo executado repetidamente.

  
let mut a = 1;

while a <= 10 {
    println!("Valor de A: {}", a);
    a += 1; //
}

O comando [break] é utilizado em laços de repetição e causa a interrupção imediata do mesmo, continuando o programa na próxima linha após o ciclo de repetição.
O comando [continue] é usado somente em laços de repetição e quando executado o laço volta imediatamente para o teste de condição do laço de repetição.

For

Usamos o comando [for] quando sabemos exatamente quantas vezes iremos repetir um bloco de instruções.

  
// 0 para 10 (excluindo o 10)
for i in 0..10{
  println!("Valor atual: {}", i);
}

// 0 para 10 (incluindo o 10)
for i in 0..=10{
  println!("Valor atual: {}", i);
}

//Utilizando arrays

let grupo : [&str; 5] = ["Juliana", "Anderson", "Christian", "Chrystian", "Thales"]


fon n in 0..grupo.len(){
  println!("Pessoa: {}", grupo[n]);
}

Match

O comando [match] nos permite comparar valores com uma série de padrões e serve para rodar frações de código condicionalmente. 

  
let tshirt_width = 20;
let tshirt_size = match tshirt_width {
    16 => "S", // checa se o tamanho é 16
    17 | 18 => "M", // checa se o tamanho é 17 ou 18
    19 ..= 21 => "L", // checa se o tamanho varia de 19 até 21 (19,20,21)
    22 => "XL",
    _ => "Indisponível",
};

println!("{}", tshirt_size); // L
