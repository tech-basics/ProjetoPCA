Funções

Uma função é um pedaço de código que executa tarefas específicas epode ser chamado a qualquer momento e quantas vezes forem necessárias no código.

  • Códigos são declarados com a palavra-chave [fn]
  • Em rust quando for passar argumentos em uma função é necessário declarar os tipos de dados
  • Por padrão, funções retornam uma tupla vazia. Se você desejar que retorne um valor o tipo do valor deve ser específicado após [->]

Olá mundo

  
fn main() {
    println!("Olá mundo!");
}

Passando argumentos

  
fn print_soma(a: i8, b: i8) {
    println!("soma é: {}", a + b);
}

Retornando valores

  
fn plus_two(a: i32) -> i32 {
    return a + 2;
}

Funções Lambda

  
fn main() {
  let x = 2;
  println!("{}", valor_quadrado(x));
}

fn valor_quadrado(i: i32) -> i32 {
    i * i
}
