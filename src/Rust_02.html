Operadores


Operadores são construções definidas em linguagensem linguagens de programação que geralmente se comportam como funções, mas que diferem sintática ou semanticamente.
Exemplos simples de operadores incluem:
Operações aritméticas ["Adição" com "+"]
Comparações ["Maior que" com ">"]
Lógicos ["Diferente" com "!""].

Operadores aritméticos


[+] - Adição
[-] - Subtração
[*] - Multiplicação
[/] - Divisão
[%] - Resto da divisão 

  
  let a = 5;
  let b = a + 4; //9
  let c = a - 2; //3
  let d = a * 3; //15
  let e = a / 2; //2 
  /*
  Os tipos devem ser mantidos
  logo o resultado não será 2.5
  */ 
  let f = 5.0 / 2.0; //2.5
  let g = a % 2; //1

Operadores de comparação


[==] - Igual
[!=] - Diferente
[<] - Menor que
[>] - Maior que
[<=] - Menor ou igual
[>=] - Maior ou igual 

  
  let a = 1;
  let b = 2;

  let c = a == b; //false
  let d = a != b; //true
  let e = a < b; //true
  let f = a > b; //false
  let g = a <= a; //true
  let h = a >= a; //true

Operadores lógicos


[!] - Diferente
[&&] - e (and)
[||] - ou 

  
  let a = true;
  let b = false;

  let c = !a; //false
  let d = a && b; //false
  let e = a || b; //true

Atibuição composta


O operador [=] é utilizado para atribuir nome a um valor ou função.
Operadores compostos são criados compondo um dos operadores vistos anteriormente com o operador [=].

  
  let mut a = 2;

  a += 5; //2 + 5 = 7
  a -= 2; //7 - 2 = 5
  a *= 5; //5 * 5 = 25
  a /= 2; //25 / 2 = 12 not 12.5
  a %= 5; //12 % 5 = 2

Fundição de tipo


Como observado nos operadores aritméticos, não é possivel fazer um float a partir de um inteiro ou vice-versa pois os tipos dos dados devem ser iguais para a operação ser executada.
Fundição de tipo trata-se de alterar o tipo de uma váriavel para realizar uma operação com ela. Para isso é utilizado o comando [as] para definir um novo tipo em uma operação.

  
  let a = 15; //i32 padrão
  let b = (a as f64) / 2.0; //7.5
